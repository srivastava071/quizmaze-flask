About QuizMaze 🎯

📖 What is QuizMaze?

QuizMaze is an interactive web-based quiz platform developed as a summer internship project to demonstrate practical applications of Data Structures and Algorithms (DSA).

The platform lets you test knowledge across DSA and Web Development, and supports user-created custom quizzes for an ever-growing library.

✨ What makes QuizMaze special?

  • 🧠 Quiz selection by category & difficulty
  • ⏱ Timed questions with auto-advance
  • 🏆 Leaderboard to track top scorers
  • 👤 Login & Register with secure auth
  • 🎨 Custom color theme picker
  • 🔐 Secure & offline-capable

🎓 Built using

A modern tech stack combining backend logic with a polished, responsive frontend.

Python Flask SQLite HTML5 CSS3 JavaScript DSA Concepts
